# Interface do Sistema

A seguir são apresentadas as principais telas desenvolvidas para o sistema de Biblioteca Virtual.

---

## Página Inicial (Home)

A página inicial é o primeiro contato do usuário com o sistema. Ela possui um botão de destaque que direciona o visitante para a tela de cadastro. Ao navegar pela página, o usuário pode visualizar o catálogo de e-books disponíveis na biblioteca.

## Cadastro de Cliente

Tela destinada ao cadastro de novos clientes. O formulário solicita informações pessoais como nome, e-mail, telefone, CPF e data de nascimento. Após essa etapa, o usuário é direcionado para a criação da senha.

</p>
